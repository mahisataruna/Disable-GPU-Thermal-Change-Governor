## Description

This is magisk module to <b>Disable GPU Thermal</b> and <b>Change GPU Governor to Performance</b>.

## Feature

- GPU Disable Thermal
- Change GPU Governor

## Requirement

- Android 10+
- Magisk v23.0 or Latest
- Snapdragon smartphone

## Installation

- Download this module.
- Open your Magisk App
- Install
- Reboot your device
